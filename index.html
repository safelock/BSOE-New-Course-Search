BSOE New Course Search

Starting Fall 2019, many BSOE Courses will have changed course numbers and codes. The content of courses will not be changing, with the exception of the new introductory Computer Science & Engineering (CSE) programming sequence. This website is to help fellow slugs quickly find new courses without having to comb through this table. This is still a work in progress so if you notice any bugs or have suggestions please email me or make an issue on github. NOTE: The only courses available for conversion so far are CMPS, CMPE, & TIM. EE and AMS Classes are supported on a beta basis. This website is not affiliated with UC Santa Cruz or any of its family of sites.
